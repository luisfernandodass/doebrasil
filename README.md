# Doe Brasil

### Objetivo

Doe Brasil é um site de arrecadação de doações para os necessitados.

### Host

Acesse o site: https://doebrasil.netlify.app/
 
### Tecnologias

As seguintes ferramentas foram usadas na construção do projeto:

- JavaScript
- HTML5
- CSS3
- Bootstrap
